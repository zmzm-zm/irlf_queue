# irlf queue
## basic information
this is a single-forward queue like:  
|_head_dummy| -> |node1| -> ... -> |nodeN| -> |_tail_dummy|  
    ^								^			|	  ^  
	|								|___________|	  |  
  |_head|										   |_tail|  
it has a picture in ~/pic/  
and it is lock-free  
ATTENTION, it doesn't have GC
